## 📸 What This Is

This is a **functional Scrum/Sprint Management Dashboard** simulating a real automotive embedded systems project (BMW LiDAR ECU / ADAS platform). It demonstrates hands-on knowledge of:

- **Sprint planning & execution** — Backlog → To Do → In Progress → Review → Done
- **Agile metrics** — Burndown charts, team velocity, story points
- **Risk & priority management** — Critical/High/Medium/Low ticket classification
- **Cross-functional team coordination** — Workload distribution per team member
- **Automotive domain language** — AUTOSAR, ISO 26262, ASPICE, CAN, ECU, OEM

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🗂 **Kanban Board** | Drag & drop tickets across sprint columns |
| 📊 **Metrics Dashboard** | Burndown, velocity chart, team workload, priority breakdown |
| 🎫 **Ticket Management** | Add, move, and detail view for each ticket |
| 👥 **Team Filter** | Filter board by team member |
| ⚡ **Live Sprint Progress** | Real-time story point tracking |

---

## 🛠 Tech Stack

- **React** (functional components + hooks)
- **Recharts** (burndown & velocity charts)
- **Pure CSS** (no UI library — all custom)
- Deployed via **GitHub Pages**
